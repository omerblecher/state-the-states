Star formula (D-11 locked):
```dart
int computeStarCount(int score, int? previousBest) {
  if (previousBest == null) return 3;           // first game ever
  if (score < previousBest) return 3;           // personal best
  if (score <= (previousBest * 1.20).ceil()) return 2;  // within 20%
  return 1;                                     // more than 20% above
}
bool _isNewPb(int score, int? previousBest) => previousBest != null && score < previousBest;
```
