    `String? stateHitTest(Offset scenePoint, List<StateData> states, {double scale = 1.0})`

    Port the Flags `hitTest()` body verbatim with these mechanical substitutions:
    - `hitTest` → `stateHitTest`
    - `CountryData` → `StateData`
    - `isoCode` → `postal`
    - Remove the entire `isDegenerate` branch from `_expandedBbox` (US states have real geometry)

    The `_expandedBbox` logic after removing `isDegenerate`:
    1. Compute `screenArea = rect.width * rect.height * scale * scale`
    2. If `screenArea < _kMinScreenArea`: return `Rect.fromCenter(center: state.centroid, width: expansionRadius * 2, height: expansionRadius * 2)` where `expansionRadius = sqrt(_kMinScreenArea / pi) / scale`
    3. Compute diagonal; if `diagonal < 1e-6` (guard for degenerate edge case): return minimal rect centered on centroid
    4. If `diagonal >= minSceneDiag`: return `rect` unchanged
    5. Otherwise: return proportionally scaled rect centered on centroid

    The `max` import is carried from Flags but verify it is actually used; if not, drop it to avoid lint warnings.
