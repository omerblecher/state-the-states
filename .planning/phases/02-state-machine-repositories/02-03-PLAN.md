<objective>
Build the `GameSessionNotifier` AsyncNotifier state machine and the
`GameLifecycleObserver`. The notifier ports the Flags analog but encodes the
three load-bearing Phase 2 deltas: (1) `Stopwatch`-as-truth elapsed with the
ticker as a display-only pulse (D-02/D-12) — `pauseGame()` calls
`_stopwatch.stop()`; (2) explicit `_hintPenalty` rehydrated from
`loadSession()` rather than back-calculated (D-05); (3) `restoreGame()` lands
in `GamePhase.paused` with the Stopwatch stopped (D-09). The observer is the
thin Flutter-binding glue (no Flags analog) that calls `pauseGame()` on
`.paused`/`.hidden` only — never `.inactive` (D-10/D-11). It is built and
widget-tested here but mounted to a screen only in Phase 4.

Purpose: This is the phase goal — all game logic in pure Dart, unit-tested
before any widget depends on it. The notifier closes SCORE-01/02 and the
SESS-01/SESS-03 logic seams; the observer closes the SESS-01 lifecycle seam.
Output: Two source files + two test files, all green.
</objective>

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-05 | Tampering / Elevation | _hintPenalty accumulation | mitigate | `_hintPenalty` is monotonically additive (only `+= 5`, never subtracted) and score is always derived, never decremented (D-06). A tamperer cannot drive score below the legitimate floor through the notifier API. |
| T-02-06 | Denial of Service (UX) | background timer accrual | mitigate | `pauseGame()` calls `_stopwatch.stop()` as its first action (D-12) so a backgrounded round accrues 0 elapsed; GameLifecycleObserver triggers this on `.paused`/`.hidden`. Without this the monotonic Stopwatch would silently inflate the player's score. Tested directly. |
| T-02-07 | Information Disclosure (COPPA) | snapshot flush payloads | mitigate | The notifier flushes via GameStateRepository.saveSession whose payload is the identifier-free field set audited in Plan 02 (T-02-04). The notifier adds no new persisted fields. |
| T-02-08 | Spoofing (walled garden) | GameSessionNotifier imports | mitigate | Notifier imports only game types, ticker, and the two repositories — zero ads-module imports (COMP-03 carried from Phase 1). Verified by import grep. |
| T-02-SC | Tampering | package installs | accept | No package-manager installs in this phase — all deps declared Phase 1. |
</threat_model>
